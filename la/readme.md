# Text Processing Learning Activity

Follow along with the videos as you learn about text processing and process management in bash.


At the end of the videos you should have the following files in the la folder

 - user_count.sh
 - change_shell.sh
 - word_count.sh

You should also have the following branches in your repository

 - main
 - user_shell_count
 - change_shell
 - word_count
